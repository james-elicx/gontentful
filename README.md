# Gontentful

Minimal Contentful client library for Go with a command line interface.

## CLI:

Install:

```sh
$ go get -u github.com/moonwalker/gontentful/cmd/gfl
```

## Dependencies

Using Go modules:

```sh
$ go mod vendor
```
