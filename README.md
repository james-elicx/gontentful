# Gontentful

Minimal Contentful client library for Go, and command line interface.

## CLI:

Install:

```sh
$ go get -u github.com/moonwalker/gontentful/cmd/gfl
```

## Dependencies

Go vendors managed with Dep:

```shell
$ dep ensure
```
